# Hacktoberfest 2019

## How to contribute

0. Register for [Hacktoberfest 2019](https://hacktoberfest.digitalocean.com)
1. Create a new folder with your github username
2. Push README.md to this folder and modify it N-times
3. Enjoy!!!
